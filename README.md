This is the same game as the Sokoban project already on my GitHub. If you haven't checked that one yet, I recommend doing so before looking at this automatic version. This version works with pre-defined patterns and records the actions played to generate a new movement file if a solution for the map is found.

To select a level, youy have to type the full name of the level, for exemple if you want to play the level 1, you have to type : "niveau1.sok", and same for the movements files : "niveau1_1.dep".
